MediScan AI

AI-Powered Healthcare Prescription Analyzer

Introduction

MediScan AI is a comprehensive healthcare prescription analyzer built with modern web technologies. It leverages AI to extract information from medical prescriptions, provide explanations, and offer health suggestions. The application features seamless authentication, file storage, OCR capabilities, and personalized health insights.

Tech Stack

React TypeScript React Router v7 Puter.js Tailwind CSS Vite Zustand PDF.js

Features

Easy Authentication

Browser-based authentication using Puter.js, no backend required.

Prescription Upload & Storage

Securely upload and store medical prescriptions (images/PDFs).

OCR Text Extraction

Automatically extract patient details, medicines, dosages, and instructions.

AI Medicine Explanations

Get detailed explanations of medications, side effects, and interactions.

Health Suggestions

Receive personalized diet plans, exercise recommendations, and lifestyle tips.

Smart Alerts

Medicine reminders, refill notifications, and follow-up alerts.

History Tracker

Save and track previous prescriptions with improvement over time.

Responsive UI

Clean, modern interface that works across all devices.

Installation & Setup

git clone https://github.com/prakash7474/med_scan.git
cd med_scan
npm install
npm run dev

Open http://localhost:5173 in your browser.

Type Definitions

Prescription Interface

interface Prescription {
    id: string;
    patientName?: string;
    doctorName?: string;
    imagePath: string;
    prescriptionPath: string;
    feedback: Feedback;
}

Feedback Interface

interface Feedback {
    medications: { score: number; tips: Tip[] };
    dosage: { score: number; tips: Tip[] };
    instructions: { score: number; tips: Tip[] };
    sideEffects: { score: number; tips: Tip[] };
    lifestyle: { score: number; tips: Tip[] };
    healthCompliance: { score: number; tips: Tip[] };
}

Puter.js Integration

The application uses Puter.js for serverless authentication, file storage, AI capabilities, and key-value storage. This allows for a backend-less architecture with user-borne costs.

Key Functions

Authentication

Browser-based sign-in/sign-out using Puter.js auth API.

File Operations

Upload, read, and delete files in Puter's cloud storage.

AI Chat

Interact with AI models for medicine explanations and health advice.

OCR Processing

Extract text from prescription images using AI-powered OCR.

Data Persistence

Store user data and preferences using key-value storage.
